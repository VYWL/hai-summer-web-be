# HAI2023-Summer-Project-Web 백엔드 Repo

간단히 구현한 OCR & GPT API 사용 서버 💽

## 구성 언어 및 라이브러리

-   Javascript
-   ExpressJS

## 필수로 있어야 하는 것들

```sh
1. .env -> OCR_KEY, GPT_KEY, OCR_URL
2. `result.json` 파일
```

## 설치 방법

### 우선 git Clone

```sh
$ git clone https://github.com/VYWL/hai-summer-web-be
$ cd hai-summer-web-be
```

### 초기 패키지 설치

```sh
$ # 의존성 패키지들을 모두 다운받는 과정
$ npm install
```

### (local) 실행

```sh
# 8081 port로 요청대기!
$ node index.js
```
